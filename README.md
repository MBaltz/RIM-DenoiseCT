# RIM-DenoiseCT
Recurrent Inference Machine for Denoise CT scans, writed Tensorflow.

## Clone:

```bash
$ git clone --recursive -j4 
```

## Pull:
If clonned with `--recursive` parameter:

```bash
$ git pull --recurse-submodules
```

If clonned normaly:
```bash
$ git submodule update --init --recursive
```
